단축 평가 논리 계산법(Short-circuit evaluation)

  • 논리연산자를 이용해서 코드를 더 짧게 구성하는 방법

I. AND 연산자

  • 간단한 이론적 원리는
true 혹은 truthy한 값&& truthy한 값=오른쪽에 있는 값
false 혹은 falsy한 값 && truthy한 값=뒤에 오는 값을 보지도 않고 false
																		or 앞에 왔던 falsy한 값

따라서

'hello' && 'bye' ==> 'bye'
1 && 'bye'==>'bye'

null && 'hello' ==> null
undefined && 'hello' ==> undefined
' ' && 'hello' ==>' '
0 && 'hello'==>0
NaN && 'hello'==> NaN

위와 같은 일종의 틀에 기인한다!

즉,

const dog={
    name:'멍멍이'
};

function getName(animal){
    if(animal){
        return animal.name;
    }
    return undefined;
}
const name=getName(dog);

는 위의 방법을 토대로 한다면,

const dog={
    name:'멍멍이'
};

function getName(animal){
   return animal && animal.name;
}
const name=getName();
console.log(name);//undefined

와 같이 간결하게 작성할 수 있다!

  • 특정 값이 유효할 때에만 어떤 값을 조회할 때 사용!
const object = null;
const name   = object && object.name;
console.log(name);//null

  • 이 문법은 나중에 React 에서 조건부 렌더링에서 사용될 수 있으니 잘 숙지하기!

II. OR 연산자

  • 어떤 값이 없으면 대체제로 어떤 것을 이용하고자 할 때 사용
  • 작동 원리
falsy한 값 || truthy 한 값 = truthy한 값
truthy한 값1 || truthy한 값2 =truthy한 값1//즉, 뒤에값을 안봄

따라서
'' || '이름없다' ==>'이름없다'

const namelessDog={
    name:''
};

function getName2(animal){
    const name = animal && animal.name;
    return name || '이름이 없는 동물입니다';
}

const test = getName2(namelessDog);
console.log(test);
